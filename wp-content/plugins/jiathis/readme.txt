JiaThis是稳步提升网站流量和搜索引擎排名的WEB2.0工具！通过JiaThis,你网站的内容将被分享到互联网的任何角落，稳步提升您的网站流量！JiaThis是中国最大的社会化分享按钮及工具提供商。

一、安全性高，JiaThis提供7×24小时服务器安全监控，多方案时刻确保分享代码安全。

二、速度最快，分享代码JS文件全部采用CDN传输，网站用户的访问请求首先到达离他最近的加速节点服务器。

三、覆盖量最广，JiaThis目前提供面向110个社会化媒体的一键分享服务，我们统一管理和维护这些分享接口。

四、用户群最大，JiaThis拥有中国最大的分享网络，分享按钮覆盖3亿互联网人群，其中每月活跃社会化分享用户高达3500多万。

五、精确的数据统计，JiaThis提供业内最精准，最全面的数据分析，让数据成为您网站优化的重要依据。

六、分享热度优化，JiaThis推出“社会化媒体排行榜”，此数据每日更新。可以通过分享热度来优化自己的分享工具。

七、智能排序功能，JiaThis依据用户使用习惯，自动进行分享媒体按钮智能排序，从而进一步调动用户对社会化信息分享的积极性和分享欲望，使网站分享量能提升到40%及以上。

八、全面的自定义功能，JiaThis客户端JS支持非常多自定义功能，详情请参见：<a href="http://www.jiathis.com/help/html/jiathis-diy" target="_blank">http://www.jiathis.com/help/html/jiathis-diy</a>。

= 我必须要在JiaThis网站注册会员才能使用吗？ =
不是的，您无需注册即可使用。但是注册用户可以享用更多额外功能，如免费享用流量统计分析，拥有独立系统管理后台等等。

== Installation ==
1. 将下载后的压缩包解压，得到一个名为jiathis文件夹（包括一个.php文件及一个.txt文件），将该文件夹上传到wordpress的 '/wp-content/plugins/'目录下，然后在WordPress后台插件菜单下激活即可。

2. JiaThis分享插件安装完成后，会自动在在日志和静态页面正文后添加JiaThis分享按钮，如果您想在其他页面添加按钮，你可以使用Widget来添加。

3、JiaThis分享插件默认嵌入的是“大图标”代码。如果您想更换其他按钮风格或希望对网站的分享数据进行追踪与分析，请到JiaThis官网<a href="http://www.jiathis.com/register" target="_blank">免费注册</a>，并在登录状态下获取新代码。然后在Wordpress后台的 “设置->JiaThis分享工具” 中将您的代码填入文本框中并提交即可。

(如果之前本版本之前安装过1.0版本的JiaThis分享工具，请卸载后重新安装本版本)
